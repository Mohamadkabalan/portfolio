ECOMMERCE PLATFORM

2ACommerce is a comprehensive e-commerce platform that enables individuals and businesses to create and manage their own online stores. It offers a wide range of tools and features to help users sell products directly to their customers across multiple channels, including web, mobile, in-person, and through social media platforms. 2ACommerce simplifies the process of setting up an online store by providing customizable website templates, a secure shopping cart, product management, inventory tracking, payment processing options, and integration with various shipping services. Beyond basic store functionalities, 2ACommerce also offers advanced features such as marketing tools, analytics to track sales and customer trends, and apps to extend the store's capabilities. It offers also PoS, digital marketing and data tools , it is a custom application integrations for the eCommerce and POS system , It is developped using wordpress , Where I have used woocommerce plugin, payment api’s integrations, shipment api’s integrations , wordpress rest api’s and i have created a custom plugins inside of it.

2ACommerce contain a wordPress inventory sync plugin which is a custom wordPress plugin that sync products and inventory from an API

Timber was used in developping 2ACommerce because it helps create fully-customized WordPress themes faster with more sustainable code. With Timber, I was able to write the website HTML using the Twig Template Engine separate from the PHP files. which helps the theme to have a clean code.

I have also used bedrock which is an ideal starting point for using Composer with WordPress.It helps Manage WordPress install and plugins with Composer, a PHP dependency manager. Composer will make development more reliable, help with team collaboration, and it helps maintain a better Git repository.

Gutenberg also is used to build content layouts using blocks, which can be customized and reused. The main purpose for using Gutenberg is for creating custom blocks and extending the functionality of WordPress through plugins like Advanced Custom Fields (ACF) Pro. Gutenberg and ACF Pro together offer a powerful platform for developing custom, dynamic content blocks in WordPress. By leveraging JavaScript and PHP, alongside the rich set of features provided by Gutenberg and ACF Pro, I wan able to create sophisticated content structures that are easy for end-users to interact with and manage, enhancing the overall experience of building and managing WordPress sites.
